# 猴子防御塔
## 简介
猴子被分为4大类（主要、军事、魔法、支持），以及随着知识系统解锁的特殊猴子（目前包括六级猴和黑色神殿）。

猴子每次被使用都会获得经验值，可以解锁更多升级。经验值为每种猴子分开计算。每种猴子有15种升级。在单人模式里面同一种T5炮塔只可拥有1个，除了xx5弩神在双弩大师（Master Double Cross）知识解锁并开启后可以同时建造2个。
