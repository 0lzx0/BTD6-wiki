# 军事流
## 简介
虽然军事猴子成型以后的攻击力上限不高，但是在climps中也能有春天！

军事流和太阳永啤流相比，能够通过小飞机、直升机或迫击炮将火力覆盖到全地图。太阳永啤流对于一些拐角少的地图还是比较吃力的。

## 猴子塔配置
- 英雄萨乌达
	- 萨乌达在1级就自带迷彩识别，而且只需要$650，非常适合在没有水面的情况下扛前期局势
- 220海盗船
	- 220是非常强的前期猴子塔
	- 4xx和5xx也能在中后期提供不俗的伤害
- 520迫击炮
	- 
- 203直升机
- 020猴村
