# 永钉流
## 简介
永钉流主要依靠`强力塔+250钉子工厂`的搭配。所以会出现很多的搭配。

战术核心是400炼金+2xx猴村+040工程师，放在道路最末尾收尾。

## 简单示例
因为永钉流的特殊性，这里只简单提供一个依靠502法师猴作为核心的打法。

R40：301忍者 -> 英雄 -> 300炼金 -> 401忍者 -> 302魔法师 -> 402炼金  
R80：猴村胶水猴 -> 402法师猴 -> 502法师猴  
R100：025钉子工厂 -> 024 回旋镖猴
