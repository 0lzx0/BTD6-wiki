# 波次 / 重点回合
## 介绍
波次是塔防游戏中的重要概念，游戏设计者通过波次来控制敌人的强度和经济的总量。对于游戏爱好者而言，如果想要成为资深的、进阶的玩家，必须对波次有清晰的认识。
