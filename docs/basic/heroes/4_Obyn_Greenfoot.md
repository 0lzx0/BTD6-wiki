# 奥本·格林福特（Captain Churchill）
## 英雄简介
`奥本·格林福特`拥有自然之力，可以用他的灵狼攻击穿过坚固的障碍物。

## 英雄属性
**经验值获取**

`奥本·格林福特`每回合会自动获得定额的经验值。

他的练级速度为 1 倍。

**体积**  
占地范围7
