# 艾蒂安(Etlenne)
## 英雄介绍
  这位高科技英雄可以用他的遥控无人机随时随地追捕气球，在版本20.0更新中初次登场。
  他的主要攻击方式是操控无人机进行远程攻击。随着等级的提升，他将能够控制更多的无人机，并为他的无人机使用越来越多的控制模式。
